PROJECTS



XML to C_struct

Designed and developed a module to represent data in XML format to a representation consist of structures and arrays as defined in the C programming language. It is implemented using C, along with flex and bison for parsing the XML using its grammar. For the transformation from XML to C structures and array(in the form of defining and initializing), I leverage the parse tree and an intermediate tree representation of the XML. This module helps in handling XML represented data while working with an application written in C.

Bird Species Classification

Developed a Convolutional Neural Network model, inspired from VCC-16 and Alex-Net, to classify birds as per species. We choose 20 birds categories out of 200 birds categories listed in the Caltech-UCSD Birds 200 (CUB-200). We compared our model with state-of-the-art Neural network models to fine-tune our parameters. During training, we used both pre-trained imageNet weights and random weights to initialise the networks. Finally, we developed a model that was at par with ResNet-50 in terms of testing accuracy and outperformed it in terms of training accuracy. We also learned the training graph through fine-tuning in order to obtain a better understanding of how neural networks works.
